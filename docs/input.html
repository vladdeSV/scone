    • hasControlKey

      Declaration

      auto hasControlKey(SCK ck);

      Check if the InputEvent has a "control key" pressed.

      Return Value

      true, if all control keys entered are pressed

      Example:

      1. foreach(input; getInputs()) { if(input.hasControlKey(ControlKey.CTRL | ControlKey.ALT | ControlKey.SHIFT)) { //do something if CTRL, ALT and SHIFT are held down... } }

    • controlKey

      Declaration

      @property auto controlKey();

      Get control all keys. Used to check if ONLY specific control keys are activated.

      Return Value

      SCK (enum)

      Example:

      1. foreach(input; getInputs()) { if(input.controlKey == ControlKey.CTRL | ControlKey.SHIFT) { //do something if only CTRL and SHIFT are held down... } }

  • InputSequence

    Declaration

    struct InputSequence;

    Wrapper for an input sequence sent by the POSIX terminal An input from the terminal is given by numbers in sequence

    Discussion

    For example, the right arrow key might send "27, 91, 67", and will be stored as [27, 91, 67]

    when on posix, a list of keypresses is loaded and used
